# Local Development

This guide is for developers running the project locally for development and testing.

## Prerequisites

- Node.js
- Python
- Docker (recommended for Redis/Postgres)

## Running locally

1. Start Postgres + Redis
2. Run the Next.js interface
3. Run the bot API
4. Run the worker

## Contributing

- Keep changes small and focused
- Prefer reusable/extensible code
- Add tests when behavior changes
